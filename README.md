# CRM-Simultaneo
Oque precisamos fazer para ter o site no ar?
Plataforma	Para que serve
GitHub	Armazenar e controlar o código do sistema.
Render	Hospedar o site, o backend e a API do sistema.
PostgreSQL	Armazenar os dados de clientes, projetos, propostas, usuários e contratos.
Cloudflare	Gerenciar o domínio e proteger o sistema contra ataques, acessos suspeitos e DDoS.
Registro.br ou outro provedor de domínio	Registrar o endereço oficial do sistema, como astra.com.br.
Cloudflare R2 ou outro armazenamento	Guardar PDFs, contratos, planilhas, imagens e outros anexos.
Resend ou outro serviço de e-mail	Enviar recuperação de senha, convites e notificações do sistema.
Recursos feitos no próprio sistema
Criptografia: protege CNPJ, CPF e outros dados sensíveis antes de salvá-los no banco.
Autenticação: controla login, senha e permissões de usuários.
Backup: protege os dados em caso de erro, exclusão ou falha.
